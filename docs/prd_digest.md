# PRD Digest — VoiceSaju

## Goals
- 1년: 가입 50,000명 / 결제 전환율 8% / 구독 500명 / DAU/MAU 0.15 / 매출 5~15억원.
- 3개월: 가입 5,000명 + 결제 200건 + 톤 검증 인터뷰 합격선 통과.
- 빈 시장 포지션("대화형 음성 + 매운맛 톤")으로 한국 1위 음성 사주·타로 서비스 진입.

## Target User
20-30대 여성 메인(P1 직장인 효주 28세 + P2 럭키맥싱 미주 24세 + P3 워킹맘 지원 33세 보조).

## Must-have Features
1. 단계별 온보딩 — 생년월일/시각(모름 옵션)/성별/이름(옵셔널) 입력.
2. 사주 풀이 흐름 — 카테고리 선택 → 인트로 15초 → 페이월 → 본 풀이 1~2분 + 꼬리질문 3개×30~40초.
3. 데일리 타로 — 매일 1장(22장 메이저), 결정적 시드, 30~40초 음성 풀이.
4. 명대사 카드 — 풀이 후 자동 생성, 카테고리별 색상, 서버 OG 1080×1920 공유.
5. 결제·구독 — 무료(타로 주1회+가입1회) / 단건(4.9~7.9천원, 꼬리질문 3개 포함) / 구독(9.9~14.9천원).
6. 회원/비회원 분기 — 비회원 첫 풀이 체험 → 가입 시 영구 저장, 카카오/애플(웹) + 토스ID(인앱).
7. 마이페이지 — 사주 명식 시각화, 풀이 히스토리, 결제/구독 관리, 사주 수정 2회 무료.
8. 자체 사주 명식 계산 엔진 + LLM 해석 분리 (환각 방지).

## Key NFRs
1. 인트로→본 풀이 첫 음성 3초 이내, TTS 첫 청크 1.5초 이내, 꼬리질문 답변 첫 음성 2초 이내.
2. 풀이 1회당 LLM+TTS 비용 ≤ 단건가의 20% (본 풀이 + 꼬리질문 3개 합산).
3. 생년월일·시각 DB 컬럼 단위 AES-256 암호화, 약관 "오락 목적" 면책 명시.
4. WCAG 2.1 AA, 자막 동시 표시, 모바일 우선 반응형 (토스 WebView + 웹).
5. 톤 가드레일 — 시스템 프롬프트 + 평가셋 50~100개 회귀 + 실시간 욕설/혐오 필터.
6. 오류 처리 — LLM 실패 시 자동 환불/토큰 적립, TTS 실패 시 자막 fallback, 네트워크 끊김 시 이어듣기.

## Scope Boundaries

### In (v1)
- 웹 + 토스 인앱 (단일 Next.js 코드베이스), 카테고리 3개(연애/직장/금전), 캐릭터 2명 고정(시니컬 누님 + 노인 도사), 결제(단건+구독), 명대사 카드 공유.

### Out (v2+)
- STT(유저 음성 입력), 모바일 네이티브 앱, 푸시 알림/카톡 알림톡, 캐릭터 선택, 다중 카드 스프레드.
- 추가 카테고리(건강/대인관계/학업), 궁합/연간운세, 꼬리질문 자유 텍스트, 답변 후 추가 꼬리질문.
