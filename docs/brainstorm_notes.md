# Brainstorm Notes — VoiceSaju (가칭)

작성일: 2026-05-28

## 1. 한 줄 요약

> **"실제 역술가와 대화하는 느낌의 보이스 사주/타로 풀이 서비스. 20-30대 타겟, 매운맛 + 요즘 세대 톤."**

---

## 2. Problem Space

### 페인 포인트 (가설)
- 기존 사주/타로 서비스는 **일방향 텍스트**라 재미가 떨어진다.
- 기성 풀이의 톤이 **꼰대스러워서** 20-30대에게 공감되지 않는다.
- **매일 볼 거리(retention 후킹)** 가 약하다.

### 타겟 유저
- **20-30대** (특히 토스 인앱·기존 사주앱 사용자)
- 사주/타로에 호기심은 있지만 기성 채널의 톤·UX에 만족 못 하는 층

### 핵심 가설
- **"대화처럼 풀어준다 + 매운맛 톤 + 진짜 역술가 보이스"** 조합이면 차별성이 충분히 확보된다.
- **데일리 타로 주 1회 무료 → 구독 유도** 의 retention loop가 만들어진다.

---

## 3. Existing Landscape

| 카테고리 | 대표 서비스 | 한계 |
|---------|-----------|------|
| 사주 앱 | 점신, 포스텔러, 헬로우봇 | 텍스트 일방향, 톤이 정통/꼰대 |
| 운세 콘텐츠 | 인스타 사주 계정, 유튜브 | 개인 맞춤 X |
| 토스 인앱 운세 | 토스 인앱 내 사주 미니앱 | 보이스/대화형 없음 |

→ **"대화형 + 매운맛 + 보이스"** 조합은 비어있다.

---

## 4. Solution Direction

### 4.1 핵심 컨셉
- STT → LLM → TTS **풀스택 양방향 음성**은 허들이 높다고 판단 (마이크 권한·환경·정확도·비용).
- MVP는 **텍스트 입력 + LLM + TTS 보이스 출력 + 꼬리질문 대화**.
- **STT(유저 음성 입력)는 v2 옵션 토글.**

### 4.2 두 가지 콘텐츠 축
1. **사주 풀이 (본격)** — 사용자가 카테고리 선택 → 1~2분 음성 풀이 + 꼬리질문 2~3개
2. **데일리 타로 (가벼움)** — 매일 1장 자동 뽑기 + 30~40초 음성 풀이

### 4.3 캐릭터 (서비스의 영혼)
- **타로 도사**: 신비로운 노인 — 데일리, 가벼운 톤
- **사주 누님**: 시니컬한 여성 — 본격 풀이, 매운맛 톤

> 둘 다 1명 고정 (캐릭터 선택 기능은 v2+).

### 4.4 사주 정확도 — 룰 기반 + LLM 분리
- **자체 사주 명식 계산 엔진** (오픈소스 만세력 라이브러리 활용; 후보: `korean-lunar-calendar`, `manseryeok` 등 PRD에서 비교)
- **LLM은 해석·매운맛 톤만 담당** (명식은 절대 LLM에 맡기지 않음)
- → 환각 방지 + 매번 일관된 명식 + 자유로운 톤

### 4.5 카테고리 (v1)
- 연애 / 직장 / 금전

### 4.6 온보딩 입력
- 생년월일 (양력/음력 선택)
- 태어난 시각 (**"모름" 옵션 지원**)
- 성별
- **"시각 모름" 모드**: 시주 없이 일간·월지 위주 풀이, 안내 멘트로 정확도 한계 고지 ("시간을 모르면 큰 줄기는 보지만 디테일은 흐릿해")

### 4.7 데일리 타로 흐름
- 매일 **1장 단일 카드** (스프레드 없음)
- 카테고리 분리 없음 — "오늘의 카드" 하나
- **결정적**: `(날짜 + 유저ID)` 시드 → 새로고침해도 같은 카드
- 풀이 길이 30~40초

## 5. 비즈니스 모델 — 하이브리드 (벤치마크: 헬로우봇형)

| 티어 | 내용 | 잠정 가격 |
|------|------|----------|
| **무료** | 데일리 타로 주 1회 + **신규 가입 시 사주 1회 무료** | — |
| **단건** | 사주 1회 결제 | 4,900~7,900원 |
| **월 구독** | 사주 월 1회 + 데일리 타로 **매일 1회** | 9,900~14,900원/월 |

- 결제 수단: 토스 인앱 = 토스페이, 웹 = 카카오페이/토스페이
- **시장 포지셔닝**: 마이타로 20,900원/월, 헬로우봇 26,000원/월 대비 **절반 수준**으로 진입 → "이 정도면 해볼까" 가격대
- **신규 가입 1회 무료 사주** = 후킹 장치 (포스텔러 벤치마크)
- **이탈 방지**: 헬로우봇이 "과한 페이월"로 이탈 유발한 사례 → 데일리 타로 주 1회 + 가입 1회 무료로 완화
- **단건 → 구독 자연 유도**: 단건 가격(4,900~7,900원) × 2회 ≈ 구독 가격 → 두 번 살 거면 구독이 이득

### 벤치마크 참고

| 앱 | 모델 | 가격 |
|----|------|------|
| 헬로우봇 | 구독 + 코인 하이브리드 | 월 26,000원 |
| 마이타로 | 구독 only | 월 20,900원 / 연 175,000원 |
| 포스텔러 | 포인트 단건 + 가입 1회 무료 | 패키지별 상이 |
| 점신 | 단건 (사람 상담) | 첫 3분 무료, 사전결제 25% 할인 |

## 6. 인증 / 유저 식별

**채널별 분리 전략**
- **토스 인앱**: 토스 ID 자동 연동
- **웹**: 카카오/애플 소셜 로그인 (사주 데이터 영구 저장 + 결제 필요)
- 비회원 1회 체험 허용 여부는 PRD에서 결정 (전환 funnel 설계 의존)

---

## 7. 바이럴 / 공유 (v1 포함)

- 풀이 종료 시 **명대사 카드** 자동 생성 (인스타 스토리 사이즈)
- 매운맛 한 줄 + 캐릭터 일러스트
- 토스: 카톡 공유 / 웹: 이미지 다운로드 + 링크 공유
- 매운맛 + 캐릭터 IP = 자연 캡쳐/공유 트리거

## 8. 플랫폼 우선순위

1. **웹** (v1)
2. **토스 인앱** (v1) — 결제·인증 API 제약 사전 검토 필요
3. **모바일 네이티브 앱** (v2+, 우선순위 낮음)

---

## 9. MVP 범위 (v1)

### 포함
- 텍스트 입력 + 보이스 출력
- 사주 풀이 (연애/직장/금전 3개 카테고리)
- 데일리 타로 (단일 카드, 결정적)
- 캐릭터 2명 고정 (타로 도사 / 사주 누님)
- 자체 사주 명식 계산 엔진 + LLM 해석 분리
- 결제 (단건 + 월 구독)
- 명대사 카드 공유

### 제외 (v2+)
- 유저 음성 입력 (STT)
- 모바일 네이티브 앱
- 푸시 알림 / 카카오 알림톡
- 캐릭터 선택 (1명 고정으로 시작)
- 추가 카테고리 (건강/대인관계/학업 등)
- 다중 카드 스프레드 (3장/켈틱 크로스 등)
- 궁합 / 연간운세 / 합방 풀이

## 11. 핵심 리스크

| 리스크 | 영향 | 완화책 |
|--------|------|--------|
| LLM이 매운맛 톤을 못 유지 / 욕설로 빠짐 | 톤 일관성·정책 | 시스템 프롬프트 + 톤 평가셋 + 가드레일 |
| TTS 보이스가 "역술가 같음"을 못 살림 | 차별성 실패 | 공급자 비교 + 캐릭터 보이스 튜닝 |
| 사주 풀이가 매번 달라 신뢰 하락 | 신뢰성 | 명식 결정적 + 해석 시드/캐싱 |
| 토스 인앱 결제·인증 제약 | 출시 지연 | PRD에서 토스 정책 최우선 확인 |
| Retention loop 약함 (주 1회 무료 후킹 부족) | 구독 전환율 | 알림 v2, 무료 횟수 A/B 옵션 |
| 매운맛 톤이 일부 유저에게 불쾌함 | 이탈 | 톤 강도 슬라이더(v2) 고려 |
