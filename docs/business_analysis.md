# Business Analysis — VoiceSaju

작성일: 2026-05-28
참고 문서: `docs/brainstorm_notes.md`

## 1. Executive Summary

> **VoiceSaju는 1조 4,000억원 한국 점술 시장에서 "대화형 음성 + 매운맛 톤" 차별점으로 20-30대를 노리는 하이브리드 BM 서비스.**

**결론: 🟢 Go (3가지 조건부)**

1. **MVP 출시 전 매운맛 톤 검증** (20-30대 5명 정성 인터뷰)
2. **TTS 비용 시뮬레이션** — 풀이 1회당 cost ≤ 단건가의 20%
3. **토스 인앱 정책 사전 확인** — 결제·인증·콘텐츠

**시장 진입 타이밍 양호**: MZ 럭키맥싱 트렌드(+39%) + AI 챗봇 시장(+41%) + 보이스 트렌드 가속이 동시에 일어남.

## 2. Market Analysis

### 2.1 시장 규모

| 단계 | 정의 | 추정 |
|------|------|------|
| **TAM** | 한국 점술 시장 전체 | **약 1조 4,000억원** (혁신의숲) |
| **SAM** | 디지털/앱 점술 시장 | **~1,500~2,800억원** (점신 830억 + 포스텔러 100억+ + 헬로우봇 등) |
| **SOM** | 3년 도달 가능 | **1년 5~15억 / 3년 30~50억** (디지털 점술 시장 1~3% 점유) |

### 2.2 시장 수요 지표

- **19-39세 74% 가 운세 소비** (5천명 설문)
- 여성 **69.2%** > 남성 51.3% (17.9%p 차이)
- 20대 68%, 30대 67.5% — 매우 균등
- 42.6% 모바일 플랫폼, 25.2% 오프라인
- **1인당 평균 8만 300원/년** 지출
- 포스텔러 MAU **62.8만 (전년 +28.2%)**, 점신 +16.6%

### 2.3 핵심 트렌드

- **럭키맥싱**: 카톡 선물하기 "액막이 명태" 키워드 상품 수 2024 대비 **39% 증가**
- **사주 = MBTI화**: Z세대에게 자기탐색·대화 소재로 소비 (자기이해 도구 관심 89%)
- **AI 챗봇 시장 +41%** (NIA 2025), 음성 챗봇 옴니채널화
- 유튜브 사주풀이 = 이미 입증된 킬러 콘텐츠 카테고리
- 청년층 심리: 취업난·경제난 → 불확실성 해소 → "사주는 가벼운 상담·대화 소재"

## 3. Competitive Landscape

### 3.1 경쟁사 비교

| 앱 | 강점 | 약점 | 우리 차별점 |
|----|------|------|-------------|
| **점신** (테크랩스) | 매출 830억, 누적 1,900만, 사람 상담 보유 | 톤 정통·꼰대, 일방향 | 매운맛+대화+보이스 |
| **포스텔러** | 누적 가입자 860만, MAU 62.8만, 단건 친화 | 보이스 없음, 일방향 | 보이스+대화 |
| **헬로우봇** | 챗봇형 텍스트 대화, 다 캐릭터, 누적 다운 1위 | 텍스트만, **"페이월 과함" 이탈 시그널** | 보이스+가벼운 가격 |
| **마이타로** | 단순 구독 모델 | 차별성 약함 | 모든 면 |

### 3.2 포지셔닝

```
┌─────────────────────────────┐
│  보이스 + 대화 + 매운맛       │ ← 우리 (빈 시장)
├─────────────────────────────┤
│  텍스트 + 대화 (헬로우봇)     │
│  텍스트 + 일방향 (포스텔러)   │
│  사람 상담 (점신)             │
└─────────────────────────────┘
```

> **"대화형 음성"은 시장에 없는 포지션.** 매운맛은 헬로우봇 일부 캐릭터에 있지만, 음성으로 듣는 매운맛은 없음.

### 3.3 페르소나 (Top 3)

#### 🎯 P1. 직장인 효주 (28세, 여) — **메인 타겟**
- IT/마케팅 사무직, 서울, 연봉 4~5천만원
- 주 1~2회 포스텔러/헬로우봇, 신년·고민 때 추가 결제
- 페인: "포스텔러는 정통, 헬로우봇은 페이월 빡셈"
- 결제 의향: 단건 5~7천원, 구독 1.5만까지
- 매운맛 친화도: ⭐⭐⭐ 높음
- **카피**: "직장 상사가 미워? 누님이 시원하게 풀어드림."

#### 🎯 P2. 럭키맥싱 미주 (24세, 여) — **바이럴 엔진**
- 대학생/취준생, 인스타 사주 계정 다수 팔로우
- 인스타·유튜브 데일리 노출
- 페인: "내 사주 입력 귀찮음, 짧게 보고 싶음"
- 결제 의향: 단건 3.9~5.9천원 충동적
- 매운맛 친화도: ⭐⭐⭐⭐⭐ 매우 높음
- **카피**: "오늘 카드 한 장 = 인스타 스토리 1개."

#### 🎯 P3. 워킹맘 지원 (33세, 여) — **객단가 견인**
- 마케팅/영업/스타트업, 결혼 임박/초기
- 가끔 본격 사주, "이직 운" 같은 결정 도구
- 결제 의향: 단건 1만원, 구독 미온적
- 매운맛 친화도: ⭐⭐ 중 (재미보단 신뢰)
- **카피**: "정통 명식 + 누님의 시원한 풀이."

> **포지셔닝 전략**: P1·P2 메인 / P3 보조. P2가 캡쳐 공유로 P1 유입을 만들고, P3가 객단가를 끌어올림.

## 4. Business Model

### 4.1 가격 구조 (하이브리드 — 헬로우봇형)

| 티어 | 내용 | 잠정 가격 |
|------|------|----------|
| **무료** | 데일리 타로 주 1회 + 신규 가입 시 사주 1회 무료 | — |
| **단건** | 사주 1회 결제 | 4,900~7,900원 |
| **월 구독** | 사주 월 1회 + 데일리 타로 매일 1회 | 9,900~14,900원/월 |

- 결제: 토스 인앱 = 토스페이 / 웹 = 카카오페이·토스페이
- **시장 포지셔닝**: 마이타로 20,900원, 헬로우봇 26,000원 대비 **절반 수준**
- **단건 → 구독 자연 유도**: 단건 2회 ≈ 구독 1회 가격

### 4.2 Go-to-Market 전략

#### 1. 토스 미니앱 — 광범위 노출
- 3,000만 유저 풀, 토스페이 원클릭
- 데일리 타로 = 진입 후킹 (30초)
- 단건 결제 위주 (구독 funnel은 무거움)

#### 2. 웹 — 바이럴·SEO
- **명대사 카드 = 핵심 바이럴 무기** (인스타·카톡 공유)
- SEO 키워드: `MZ 사주`, `매운맛 사주`, `AI 사주`, `오늘의 타로`, `사주 챗봇`
- 카카오톡 채널 운영 (헬로우봇 벤치마크)
- 구독 funnel은 웹 (카카오/애플 로그인 + 결제)

#### 3. 인플루언서 시딩 — 초기 1~3개월 부스터
- 인스타 운세 계정 5~10명: 무료 사주 + 명대사 카드
- 유튜브 사주 채널 1~2팀: 시니컬 누님 캐릭터 콜라보
- 트위터/스레드 짤 시딩

### 4.3 KPI (1년 차 목표)

| 지표 | 목표 |
|------|------|
| 가입자 | 5만+ |
| 결제 전환율 | 5~10% (포스텔러 벤치마크) |
| 구독 전환율 | 1~2% |
| DAU/MAU | 0.15+ (데일리 타로 retention) |
| 매출 | 5~15억 |

> **가장 큰 lever**: 명대사 카드 바이럴 × 토스 미니앱 노출. 톤 검증 통과 시 인플루언서 시딩이 폭발 트리거.

### 4.4 TTS 공급자 — Supertone (결정)

- **한국어 TTS 1위** (서울대 음성 AI 연구팀 기술)
- 시니컬 누님 / 신비로운 노인 캐릭터 보이스 적합
- 구독제 (월 크레딧, 롤오버 X)
- API 요청 제한: 20~60 req/min (티어별)
- 가격 비공개 → 비즈니스 컨택 필요 (business@supertone.ai)
- **리스크**: 크레딧 소진 시 보이스 일시정지 → PRD에서 사용량 시뮬 + 적정 티어 산정

## 5. Risks & Mitigations

### 5.1 SWOT

| | |
|---|---|
| **Strengths** ✅ | 빈 시장 포지션 (대화형 음성) · 시장가의 절반 가격 · 캐릭터 IP 명확 |
| **Weaknesses** ⚠️ | 차별점 사용자 검증 X · TTS 비용 부담 · 단일 캐릭터 = 취향 안 맞으면 이탈 |
| **Opportunities** 🚀 | MZ 럭키맥싱 (+39%) · AI 챗봇 시장 (+41%) · 토스 3,000만 유저 |
| **Threats** 🚨 | 점신·포스텔러 패스트팔로워 · 앱스토어 매운맛 톤 검열 · TTS 가격 변동 |

### 5.2 Top 8 Risks

| 리스크 | 심각도 | 완화책 |
|--------|--------|--------|
| **차별점 미검증** ("매운맛+보이스" 통할지) | 🔴 High | MVP 전 톤 샘플 정성 인터뷰 (20-30대 5명) |
| **TTS 비용** (텍스트 대비 5~10배) | 🔴 High | ① 결제 콘텐츠만 음성 ② 캐싱 ③ 짧은 풀이 ④ Supertone 티어 시뮬 |
| **LLM 톤 가드레일** (욕설로 빠지면 앱스토어 정책 위반) | 🔴 High | 시스템 프롬프트 + 톤 평가셋 + 실시간 필터 |
| **점신·포스텔러 패스트팔로워** | 🟡 Med-High | 캐릭터 IP화 (시니컬 누님) + 빠른 이터레이션 |
| **토스 인앱 정책** (결제·인증·콘텐츠 제약) | 🟡 Med | PRD 단계에서 토스 공식 가이드 우선 확인 |
| **데일리 retention 약함** (주 1회 무료 후킹 부족) | 🟡 Med | 후킹 메커니즘 강화 (캐릭터 멘트, 명대사 카드, 알림 v2) |
| **단일 캐릭터 이탈** (취향 안 맞으면 즉시 이탈) | 🟡 Med | 캐릭터 보이스 톤 평가셋 다양화, v2에서 캐릭터 선택 |
| **법적 (오락 면책)** ("사주 틀렸다" 분쟁) | 🟢 Low | 약관 "오락 목적" 면책 명시 (업계 표준) |

### 5.3 톤 검증 인터뷰 설계 (Pre-MVP)

**목표**: 30분 안에 "매운맛 톤이 통하는가" 확인

**준비물**:
- Supertone 톤 샘플 클립 5~7개 (시니컬 누님 보이스)
- 비교용 기존 앱(포스텔러) 풀이 1개
- 명대사 카드 시안 3종

**대상 (5명)**: P1×2 + P2×2 + P3×1

**스크립트 (30분)**:

| 단계 | 시간 | 질문 |
|------|------|------|
| 온보딩 | 5분 | 평소 운세 채널? 마지막 결제 언제? 공유 경험? |
| 블라인드 비교 | 10분 | A(정통) vs B(매운맛) — 재미·신뢰·공유 욕구·불편함 |
| 명대사 카드 | 5분 | 스토리 공유 의향? 가장 짤 잠재력? |
| 가격 의향 | 5분 | 단건 4,900원? 7,900원? 구독 9,900원? |
| 마무리 | 5분 | 추천 멘트? 가장 큰 거부감? |

**합격선**:

| 기준 | 합격선 |
|------|--------|
| 매운맛 > 정통 톤 선호 | 3/5 이상 |
| "불편/공격적이다" | 1/5 이하 |
| 캡쳐 공유 의향 | 3/5 이상 |
| 단건 4,900원 결제 의향 | 3/5 이상 |

**Pivot/Kill 신호**:
- 매운맛 선호 2/5 이하 → 톤 강도 조정·캐릭터 옵션 분기
- "이건 욕이야" 3/5 이상 → 톤 즉시 완화
- 캡쳐 의향 1/5 이하 → 바이럴 전략 재설계

### 5.4 법규제 / 컴플라이언스

- **점술업 특별 라이선스 없음** — 일반 소프트웨어 사업과 동일
- 통신판매업 신고 필수 (결제 운영)
- 약관 "오락 목적" 면책 명시 (업계 표준)
- 앱스토어 콘텐츠 정책: 욕설/혐오 표현 사전 검열 필요
